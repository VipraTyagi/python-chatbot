# Chatbot
The project is completed as part of the course Artificial Intelligence and the Web at University of Osnabr√ºck. A chatbot created with Python and Flask.

To get started follow the steps below:

1. Install a virtual environment by runnning the following
```
virtualenv chatbotenv
source chatbotenv/bin/activate
```

2. Install all the required libraries . Please make sure to use python between 3.8 and 3.11 as tensorflow works with these python version only.
```
pip install nltk
pip install numpy
pip install keras
pip install tensorflow
pip install flask
```

Run the chatbot.py file to create the model
```
python chatbot.py
```

Run the APP to create a Flask front end on port 8888 (or any port the app is pointing to)
```
python app.py
```
